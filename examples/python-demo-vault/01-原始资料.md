### 【S1 - 01】关于文件打开与关闭
在 Python 中，使用 `open(file, mode='r', encoding=None)` 打开文件并返回一个文件对象。
必须注意：完成操作后必须调用 `file.close()` 关闭文件，否则可能导致文件描述符泄漏。

```python
f = open('workfile', 'r', encoding='utf-8')
work_data = f.read()
f.close()
```

### 【S1 - 02】使用 with 关键字
为了防止忘记调用 `close()`，推荐使用 `with` 上下文管理器。即使在块内发生异常，它也会确保文件被正确关闭。

```python
with open('workfile', 'r', encoding='utf-8') as f:
    read_data = f.read()
```
