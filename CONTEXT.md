# Mastery Learning Skills Context

本仓库技能通用的语言与领域术语词典。Agent 在教学与存储操作中必须使用以下术语，避免概念混淆。

## Language (语言规范)

**掌握度账本 (Mastery Ledger)**:
存储课程大纲和掌握状态的 Markdown 账本，即本地 `03-掌握度账本.md` 文件。
_避免_: 学习状态日志、学习记录本、状态记录

**前置依赖 (Prerequisite)**:
进入新知识点学习前必须通过的门禁，通常需要达 4 分或以上。
_避免_: 先决条件、前置课程、依赖树节点

**提示梯子 (Hint Ladder)**:
答错或卡壳时，Agent 提供的渐进式非直接提示（暗示 -> 骨架 -> 相似 worked example -> Graded task）。
_避免_: 解题思路、直接解析、答案提示

**认知修复 (Cognitive Repair)**:
掌握度评级低于 3 分时触发的诊断重测环路，用于纠正根本概念错误。
_避免_: 错题修复、错误订正、复习补课

**课程工作区 (Course Workspace)**:
保存课程原始资料、Ledger 账本及日记的物理文件夹。
_避免_: 知识库目录、保管库、Vault

**原始资料 (Source Material)**:
学生传入的、供 Agent 切分与教学的权威文本/代码源文件（标记为 `S1`, `S2` ...）。
_避免_: 参考资料、主教材、课本

## Relationships (关系)

- 每一个 **课程工作区** 包含且仅包含一个 **掌握度账本**。
- **掌握度账本** 通过 **前置依赖** 链条连接各知识点。
- 在进行教学打分时，若得分不达标，进入 **认知修复**，使用 **提示梯子** 引导学生。
